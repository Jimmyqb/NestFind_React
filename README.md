# NestFind 🏠

A modern real estate web app built with React, Tailwind CSS, and Leaflet.

## Features
- Browse property listings across Mexico & USA
- Filter by city, type, country, bedrooms, and price
- Property detail page with interactive map
- Skeleton loading states
- Fully responsive

## Tech Stack
- React + Vite
- Tailwind CSS
- React Router
- Leaflet / React Leaflet

## Getting Started
\```bash
npm install
npm run dev
\```
